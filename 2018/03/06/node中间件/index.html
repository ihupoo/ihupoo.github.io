node常用库

2018-03-06
node

文章目录

  1. node库
  2. express库

node库

  • 通用
    • pm2 : node进程管理工具，可以利用它来简化很多node应用管理的繁琐任务，如性能监控、自动重启、负载均衡等。
  • 调试/测试
    • nodemon : node调试工具，自动重启服务，开发时候很好用。
    • supertest ： 测试库，用来配合 express（准确来说是所有兼容connect的web框架）进行集成测试，属于http请求测试库。其他测试有mocha（框架测试），chai（测试结果断言库）
  • 代理
    • http-proxy-middleware : 反向代理（webpack也采用这个）。
    • whistle : 反向代理，据说很好用。
  • 数据库
    • mongoose : mongodb的odm（Object-Document Mapping ，对象文档映射），用于操作mongodb。
    • mysql : node操作MySQL的引擎，可以在node.js环境下对MySQL数据库进行建表，增、删、改、查等操作。
  • 爬虫
    • superagent : http库，可以发起get，post等请求，常用于爬虫或者代理。
    • cheerio : 网页处理库，node.js版的jquery，可用类似api实现网页中css selector取数据，常用于爬虫。
  • 其他工具
    • decimal.js : 浮点计算
    • eventProxy : 事件发布订阅工具，个人感觉有点像轻量级的rxjs，可用于解决回调金字塔（当然async也可以）。
    • node-schedule : 定时器/job任务，适合复杂的时间调度，例如每周一每小时的20分钟和50分钟做something，普通的setInterval适合每60秒执行什么任务，较为简单容易。

express库

  • cookie
    • cookie-parser : express 4.x 操作cookie。
  • session
    • express-session : 操作session。
    • connect-redis : 操作session时设置redis缓存，来实现进程间共享session。
    • connect-mongo : 操作session时,将 session 存储于 mongodb，需结合 express-session 使用。
    • connect-flash : 基于session 实现的用于通知功能的中间件，需结合 express-session 使用。

koa2库

  • 路由
    • koa-router : 路由中间件，使用后不管html文件具体在什么位置，跳转html使用路由路径。
  • 迁移工具
    • koa-convert : 对generator中间件的转换，例如 app.use(convert(json()))等等。
  • 日志
    • koa-logger : koa开发时替换console.log输出。
    • log4js-node : 需要按照时间或者按照文件大小，本地输出log文件，可以使用邮件等形式发送日志。
  • 请求
    • koa-static : 静态资源中间件,和koa-router一起用，假如请求路径重名，优先级高。
    • koa-bodyparser : 把koa2上下文的formData数据解析到ctx.request.body。xml数据可以用koa-xml-body。
    • busboy : 解析POST请求，node原生req中的文件流，用于上传文件。
    • koa-jsonp : 直接处理jsonp请求。
  • session
    • koa-session-minimal/koa-session : session中间件，提供存储介质的读写接口。
    • koa-mysql-session : koa-session-minimal中间件提供MySQL数据库的session数据读写操作。
  • 模板引擎
    • koa-views : 模板引擎，可搭配pug等使用。
  • 错误获取
    • koa-onerror : 错误信息，hack了ctx.onerror，能够在一个地方处理所有的错误。
  • token
    • koa-jwt : koa中间件，验证json web tokens模块。
    • jsonwebtoken : 生成token，可以和koa-jwt配合使用。
  • 压缩
    • koa-compress : 压缩工具，可以采用Gzip等压缩。
